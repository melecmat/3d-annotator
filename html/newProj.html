Choose model

3D model can be of type .glb, .gltf or .obj. If you choose .obj model, it will be further optimised and transformed into .glb for use in the project (this operation may take some time).

You can either select one model, or select multiple models of diferrent qualities. You should provide short description of individual qualities in each of your specified languages, for example how big they are, if older devices can handle them easily etc.
